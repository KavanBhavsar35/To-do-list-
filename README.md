## Usage

1. **Login/Signup**: If you're a new user, sign up for an account. Existing users can log in to access their tasks.

2. **Home Page**: Once logged in, you'll be directed to the main task management page.

3. **Add Task**: Click the "Add" button to add a new task. Enter the title and description of the task, and click "Add Task."

4. **Edit Task**: Click the "Edit" button to modify an existing task. Update the title or description and save the changes.

5. **Mark Done**: Mark a task as done by clicking the "Mark Done" button.

6. **Delete Task**: Remove a task by clicking the "Delete" button.

7. **Logout**: If you're done, log out from your account.
